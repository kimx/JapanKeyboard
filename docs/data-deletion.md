# Data Deletion Instructions / 資料刪除說明

**Last Updated / 最後更新：** 2025-01-01

---

## English

### Overview

**JK Japan Keyboard Helper** stores all data **locally on your device only**. No personal data is ever transmitted to or stored on external servers. Therefore, data deletion is fully in your control and can be performed at any time without contacting us.

---

### What Data Is Stored

| Data | Storage Location | Description |
|------|-----------------|-------------|
| Input history | Your device (Chrome local storage) | Japanese text you have previously typed, used to power the history feature |
| Extension settings | Your device (Chrome local storage) | Your keyboard preferences (e.g., Hiragana/Katakana mode) |

---

### How to Delete Your Data

#### Method 1 — Delete via the Extension UI (Recommended)

1. Click the **JK Japan Keyboard Helper** icon in your Chrome toolbar to open the Extension
2. Navigate to the **History (歷史資料)** section
3. Click the **clear / delete** button to remove all saved input history
4. Your settings can be reset by uninstalling and reinstalling the Extension

#### Method 2 — Delete via Chrome Settings

1. Open Chrome and go to `chrome://settings/`
2. Click **Privacy and security** → **Site settings** → **View permissions and data stored across sites**
3. Search for the Extension and select **Delete** to remove all stored data

#### Method 3 — Uninstall the Extension

Uninstalling the Extension will permanently remove all locally stored data:

1. Right-click the **JK Japan Keyboard Helper** icon in Chrome toolbar
2. Select **Remove from Chrome...**
3. Click **Remove** to confirm

All local data (history and settings) will be permanently deleted.

#### Method 4 — Clear All Chrome Extension Storage

1. Open Chrome and go to `chrome://settings/clearBrowserData`
2. Under **Advanced**, check **Hosted app data**
3. Click **Clear data**

> ⚠️ This will clear storage data for **all** your Chrome extensions and apps.

---

### No Server-Side Data to Delete

Since JK Japan Keyboard Helper does **not** collect or transmit any data to external servers, there is no server-side data associated with your account or device to request deletion of.

If you signed in to any third-party service through our related web services (not the Extension), please refer to that platform's data deletion process:

- **Google:** [https://myaccount.google.com/data-and-privacy](https://myaccount.google.com/data-and-privacy)
- **Meta / Facebook:** [https://www.facebook.com/help/delete_account](https://www.facebook.com/help/delete_account)
- **LINE:** [https://help.line.me/line/](https://help.line.me/line/)

## 繁體中文

### 概述

**JK日語小鍵盤**所有資料**僅儲存在您的本機裝置上**。任何個人資料均不會傳送至外部伺服器或儲存於其中。因此，資料刪除完全由您掌控，可隨時執行，無需聯絡我們。

---

### 儲存的資料內容

| 資料 | 儲存位置 | 說明 |
|------|----------|------|
| 輸入歷史記錄 | 您的裝置（Chrome 本機儲存） | 您過去輸入的日文，用於歷史記錄功能 |
| 擴充功能設定 | 您的裝置（Chrome 本機儲存） | 您的鍵盤偏好設定（如平假名/片假名模式） |

---

### 如何刪除您的資料

#### 方法一 — 透過擴充功能介面刪除（建議）

1. 點擊 Chrome 工具列中的 **JK日語小鍵盤** 圖示開啟擴充功能
2. 前往**歷史資料**區塊
3. 點擊**清除 / 刪除**按鈕，移除所有已儲存的輸入歷史記錄
4. 如需重置設定，可解除安裝後重新安裝擴充功能

#### 方法二 — 透過 Chrome 設定刪除

1. 開啟 Chrome，前往 `chrome://settings/`
2. 點擊**隱私權和安全性** → **網站設定** → **查看各網站儲存的權限和資料**
3. 搜尋本擴充功能，選擇**刪除**以移除所有儲存的資料

#### 方法三 — 解除安裝擴充功能

解除安裝擴充功能將永久移除所有本機儲存的資料：

1. 右鍵點擊 Chrome 工具列中的 **JK日語小鍵盤** 圖示
2. 選擇**從 Chrome 中移除...**
3. 點擊**移除**確認

所有本機資料（歷史記錄與設定）將被永久刪除。

#### 方法四 — 清除所有 Chrome 擴充功能儲存

1. 開啟 Chrome，前往 `chrome://settings/clearBrowserData`
2. 在**進階**選項下，勾選**託管應用程式資料**
3. 點擊**清除資料**

> ⚠️ 此操作將清除**所有** Chrome 擴充功能與應用程式的儲存資料。

---

### 無伺服器端資料需刪除

由於 JK日語小鍵盤**不收集也不傳送**任何資料至外部伺服器，因此沒有與您帳戶或裝置相關的伺服器端資料需要申請刪除。

若您曾透過我們的相關網路服務（非本擴充功能）登入第三方服務，請參閱該平台的資料刪除流程：

- **Google：** [https://myaccount.google.com/data-and-privacy](https://myaccount.google.com/data-and-privacy)
- **Meta / Facebook：** [https://www.facebook.com/help/delete_account](https://www.facebook.com/help/delete_account)
- **LINE：** [https://help.line.me/line/](https://help.line.me/line/)
