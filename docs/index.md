# JK Japan Keyboard Helper
# JK 日本語キーボード ヘルパー

> A free Chrome extension for Japanese text input — no physical Japanese keyboard required.
> 一個免費的 Chrome 擴充功能，讓您輸入日文，無需實體日文鍵盤。

---

## 🔧 Install / 安裝

[![Chrome Web Store](https://img.shields.io/badge/Chrome%20Web%20Store-Install-blue?logo=googlechrome)](https://chrome.google.com/webstore)

---

## ✨ Features / 功能

- 🌸 On-screen **Hiragana** (平假名) keyboard
- 🔤 On-screen **Katakana** (片假名) keyboard
- 🕘 **Input history** (輸入歷史) for quick reuse
- 🔒 All data stored **locally** — no account, no server
- 🌍 Supports **English / 中文 / 日本語**

## 🔐 Permissions / 權限

| Permission | Purpose |
|------------|---------|
| `storage` | Save input history & settings locally on your device |
| `tabs` | Detect the active tab to enable keyboard on the correct page |

> No personal data is collected or transmitted. Everything stays on your device.
> 不會收集或傳送任何個人資料，一切都保留在您的裝置上。
