# Privacy Policy / 隱私權政策

**Last Updated / 最後更新：** 2025-01-01

---

## English

### 1. Overview

This Privacy Policy describes how **JK Japan Keyboard Helper** ("the Extension", "we", "our") handles information when you use our Chrome browser extension. We are committed to protecting your privacy.

### 2. Information We Collect

The Extension collects **no personal information**. Specifically:

| Data Type | Collected? | Purpose |
|-----------|------------|---------|
| Personal identity (name, email, phone) | ❌ No | — |
| Browsing history | ❌ No | — |
| Typed text sent to servers | ❌ No | — |
| Input history (local only) | ✅ Yes | Stored **locally** on your device via Chrome's `storage` API to power the history feature |
| Extension settings/preferences | ✅ Yes | Stored **locally** on your device via Chrome's `storage` API |

### 3. How We Use Information

All data (input history and settings) is stored **exclusively on your local device** using the Chrome `storage` API. This data is:

- Never transmitted to any external server
- Never shared with third parties
- Never used for analytics or advertising
- Accessible only to you through the Extension

### 4. Permissions Explained

| Permission | Reason |
|------------|--------|
| `storage` | Save your input history and preferences locally on your device |
| `tabs` | Detect the currently active tab to enable the keyboard input feature on the correct page |

### 5. Third-Party Services

The Extension does **not** integrate with or transmit data to any third-party service, including Google, Meta, LINE, or any other platform.

If you sign in to any third-party service (e.g., Google, Meta, LINE) through our **related web services** (not the Extension itself), those platforms' own privacy policies apply:

- [Google Privacy Policy](https://policies.google.com/privacy)
- [Meta Privacy Policy](https://www.facebook.com/privacy/policy/)
- [LINE Privacy Policy](https://line.me/en/terms/policy/)

### 6. Data Retention

Your local data persists until you:

- Manually clear the Extension's history within the Extension UI
- Remove/uninstall the Extension
- Clear Chrome's extension storage data via Chrome settings

### 7. Children's Privacy

The Extension does not knowingly collect any information from children under the age of 13.

### 8. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected by updating the "Last Updated" date above. Continued use of the Extension constitutes acceptance of the updated policy.

### 9. Contact Us

If you have questions about this Privacy Policy, please contact us at:

- **Website:** [http://project.kimx.info](http://project.kimx.info)
- **GitHub Issues:** [https://github.com/kimx/JapanKeyboard/issues](https://github.com/kimx/JapanKeyboard/issues)

## 繁體中文

### 1. 概述

本隱私權政策說明 **JK日語小鍵盤**（「本擴充功能」）在您使用我們的 Chrome 瀏覽器擴充功能時如何處理資訊。我們致力於保護您的隱私。

### 2. 我們收集的資訊

本擴充功能**不收集任何個人資訊**。詳細說明如下：

| 資料類型 | 是否收集 | 用途 |
|----------|----------|------|
| 個人身份資料（姓名、Email、電話） | ❌ 否 | — |
| 瀏覽紀錄 | ❌ 否 | — |
| 輸入文字（傳送至伺服器） | ❌ 否 | — |
| 輸入歷史紀錄（僅本機） | ✅ 是 | 透過 Chrome `storage` API **儲存於您的裝置本機**，用於歷史記錄功能 |
| 擴充功能設定/偏好 | ✅ 是 | 透過 Chrome `storage` API **儲存於您的裝置本機** |

### 3. 資訊使用方式

所有資料（輸入歷史與設定）**僅儲存在您的本機裝置**，透過 Chrome `storage` API 存取。這些資料：

- 從未傳送至任何外部伺服器
- 從未與第三方共享
- 從未用於分析或廣告目的
- 僅您可透過本擴充功能存取

### 4. 權限說明

| 權限 | 原因 |
|------|------|
| `storage` | 在您的裝置本機儲存輸入歷史記錄與偏好設定 |
| `tabs` | 偵測目前活躍分頁，使鍵盤輸入功能在正確頁面運作 |

### 5. 第三方服務

本擴充功能**不整合**任何第三方服務，也不向任何第三方（包含 Google、Meta、LINE 等平台）傳送資料。

若您透過我們的**相關網路服務**（非本擴充功能本身）登入第三方服務，該平台的隱私權政策適用：

- [Google 隱私權政策](https://policies.google.com/privacy)
- [Meta 隱私權政策](https://www.facebook.com/privacy/policy/)
- [LINE 隱私權政策](https://line.me/en/terms/policy/)

### 6. 資料保留

您的本機資料將持續保留，直到您：

- 在擴充功能介面中手動清除歷史記錄
- 移除/解除安裝本擴充功能
- 透過 Chrome 設定清除擴充功能儲存資料

### 7. 兒童隱私

本擴充功能不刻意收集 13 歲以下兒童的任何資訊。

### 8. 政策變更

我們可能不時更新本隱私權政策。變更將透過更新上方的「最後更新」日期反映。繼續使用本擴充功能即表示接受更新後的政策。

### 9. 聯絡我們

如您對本隱私權政策有任何疑問，請透過以下方式聯絡我們：

- **網站：** [http://project.kimx.info](http://project.kimx.info)
